Check the status of the current development cycle.

## Usage

```
/ring:dev-status
```

## Output

Displays:
- Current cycle ID and start time
- Tasks: total, completed, in progress, pending
- Current task and gate being executed
- Assertiveness score (if tasks completed)
- Elapsed time
